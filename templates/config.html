YAML配置格式说明

配置文件使用YAML格式，包含以下主要部分：

社交媒体账号配置
social_networks:
  - type: twitter
    socialNetworkId: username
    tag: finance
    prompt: "自定义提示词..."
    bypass_ai: false
  • type: 社交媒体类型，目前支持 twitter
  • socialNetworkId: 账号ID或用户名
  • tag: 标签，用于分类和使用不同的提示词模板
  • prompt: 自定义提示词，如果不提供则使用默认提示词
  • bypass_ai: 是否绕过AI判断直接推送，默认为false
键盘快捷键
  • Ctrl + S: 保存配置
  • Ctrl + F: 查找
  • Ctrl + Space: 自动完成
  • F11: 全屏编辑
编辑账号
@
不包含@符号的用户名
用于分类和使用不同的提示词模板
  • 财经分析模板
  • 科技分析模板
  • 通用分析模板
提示词中使用 {content} 作为内容占位符
